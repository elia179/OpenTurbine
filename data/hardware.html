Safety Interlocks

⚠ Disabling safety interlocks can cause equipment damage or injury. Only disable for specific test purposes.

Overspeed
N1 exceeds RPM limit → immediate fault shutdown.
Overtemp
TOT exceeds temperature limit → fault shutdown.
Low Oil
Oil pressure below running minimum → fault shutdown.
Oil Zero
Oil pressure near zero → catastrophic leak → immediate fault.
Flameout
Flame lost for > flameout delay → fault shutdown. Relight can prevent this.
Low Fuel
Fuel flow sensor reads low → fault shutdown. Requires fuel flow sensor.
Hot Start
TOT above threshold when START pressed → abort startup. Threshold in Config → Safety Monitor.
TIT Overtemp
Turbine inlet temperature exceeds limit → fault shutdown. Requires TIT sensor + Config → TIT Limit.
Oil Temp High
Oil temperature above limit → fault shutdown. Requires oil temp sensor + Config → Oil Temp Limit.
Low Fuel Pressure
Fuel pressure drops below minimum → fault shutdown. Requires fuel pressure sensor + Config → Fuel Press Min.
Battery Undervoltage
Bus voltage below minimum → fault shutdown. Requires battery voltage sensor + Config → Battery Min V.
Surge / Instability
N1 RPM variance exceeds threshold → shutdown. Set variance threshold in Config → Surge Detect Variance. 0 = disabled.
Digital Inputs

Up to 4 configurable digital input channels. Use for pressure switches, limit switches, safety interlocks, or any switch that should trigger an action. Each channel can be labelled and assigned a role.
