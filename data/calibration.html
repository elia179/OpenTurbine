Pre-Flight Checklist — complete before first run

Check each item below. This list does not affect the ECU — it is a reminder guide only. Tick all items before pressing START for the first time on a new build or after maintenance.

  • Oil calibration done (zero, min, max pressures set)
  • Throttle calibration done (min and max ADC values set)
  • Flame sensor threshold set (threshold marker between idle noise and flame level)
  • N1 RPM limit set correctly for your engine model
  • TOT limit set correctly for your engine model
  • Oil prime test passed — oil flows and pressure is visible
  • Igniter test passed — spark confirmed
  • Fuel prime test passed — fuel flows from nozzle
  • Starter spin test passed — starter spins freely
  • All fuel and oil connections checked for leaks
  • Engine area clear of bystanders and flammable material
  • Fire extinguisher within reach
