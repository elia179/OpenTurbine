Oil Temperature (NTC or thermocouple)
Sensor type NTC = ADC thermistor (wire to ADC1 pin). Others = SPI chips (CLK/CS/MISO required).
GPIO pin (ADC1 only) Use an ADC1-capable GPIO. Wire NTC + fixed resistor voltage divider to this pin.
B coefficient (Beta) Thermistor B constant. Usually printed on the datasheet. Typical 10 kΩ NTC: 3950.
NTC resistance at 25 °C (Ω) Thermistor resistance at 25 °C. Typical hobby NTC: 10000 Ω (10 kΩ).
Pull-up resistor (Ω) Fixed resistor in the voltage divider between 3.3 V and the NTC. Match the resistor you wired in. Typical: 10000 Ω (10 kΩ).
Data pin (any GPIO) OneWire data line. Any GPIO works; add a 4.7 kΩ pull-up to 3.3 V on this line.
Resolution Higher resolution = more precise but slower. 12-bit: ±0.0625 °C, 750 ms. 9-bit: ±0.5 °C, 94 ms.

Torque Sensor (ADC, 0–3.3 V output)
Interface type ADC: sensor outputs 0–3.3 V analog signal. HX711: standard load cell bridge amplifier (24-bit, 2-wire digital).
GPIO pin (ADC1 only) Use an ADC1-capable GPIO. Most torque sensors output 0.5–4.5 V; use a divider to fit 3.3 V.
Scale (Nm / V) Sensor sensitivity in Nm per volt. Check datasheet. Example: 50 Nm/V.
Zero offset (Nm) Output at 0 V input (usually 0). Adjust if sensor has a non-zero offset voltage.
DT pin (Data out) HX711 DOUT → ESP32 input. Any GPIO (not ADC-restricted). Avoid strapping pins.
SCK pin (Clock) HX711 SCK ← ESP32 output. Any GPIO output. Avoid strapping pins.
Scale factor (Nm / raw count) Apply a known load and divide measured Nm by the HX711 count to get this factor. Example: 0.0001 for a 50 Nm sensor at 500,000 counts full-scale.
Zero count offset Raw HX711 count at zero load. Set during tare / zero calibration. Default 0.

Safety Interlocks

⚠ Disabling safety interlocks can cause equipment damage or injury. Only disable for specific test purposes.

Overspeed
N1 exceeds RPM limit → immediate fault shutdown.
Overtemp
TOT exceeds temperature limit → fault shutdown.
Low Oil
Oil pressure below running minimum → fault shutdown.
Oil Zero
Oil pressure near zero → catastrophic leak → immediate fault.
Flameout
Configured flameout source is lost for > flameout delay → fault shutdown. Relight can prevent this.
Low Fuel
Fuel flow sensor reads low → fault shutdown. Requires fuel flow sensor.
Hot Start
TOT above threshold when START pressed → abort startup. Threshold in Config → Safety Monitor.
TIT Overtemp
Turbine inlet temperature exceeds limit → fault shutdown. Requires TIT sensor + Config → TIT Limit.
Oil Temp High
Oil temperature above limit → fault shutdown. Requires oil temp sensor + Config → Oil Temp Limit.
Low Fuel Pressure
Fuel pressure drops below minimum → fault shutdown. Requires fuel pressure sensor + Config → Fuel Press Min.
Battery Undervoltage
Bus voltage below minimum → fault shutdown. Requires battery voltage sensor + Config → Battery Min V.
Surge / Instability
N1 RPM variance exceeds threshold → shutdown. Set variance threshold in Config → Surge Detect Variance. 0 = disabled.
Digital Inputs

Up to 4 configurable digital input channels. Use for pressure switches, limit switches, safety interlocks, or any switch that should trigger an action. Each channel can be labelled and assigned a role.
