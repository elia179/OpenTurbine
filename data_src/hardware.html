System Profile
Profile ID / WiFi SSID Name of this engine. Used as the Wi-Fi AP SSID. Default: OpenTurbine. ⚠ Longer than 32 bytes — WiFi limits SSIDs to 32 bytes, so the broadcast AP name will be truncated (the full ID is still used everywhere else).
WiFi AP Password Optional password for the Wi-Fi access point. A saved password is retained unless you enter a replacement; blank selects an open network. ⚠ On an open network, anyone who joins the AP can send engine commands — set a password for anything beyond bench use.

MAVLink Telemetry Output
MAVLink v1 Serial Output
Streams engine data as MAVLink HEARTBEAT + NAMED_VALUE_FLOAT packets. Compatible with Mission Planner, QGroundControl, and similar GCSs. Uses UART2 (TX only). Does not interfere with the cluster serial port.
TX pin (GPIO) TX output only — no RX needed. Avoid GPIO 1 (UART0 TX).
Baud rate Must match the receiving display / ground-station serial setting, or there is no communication.
Interval (ms) How often to send a telemetry packet. 100–500 ms typical.
