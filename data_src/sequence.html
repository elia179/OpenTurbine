Sequence Editor

Edit startup and shutdown sequences. Each block shows its control structure, condition, and configurable parameters. Changes to sequence order require Hardware → Save & Reboot. Changes to parameters only apply after Config → Save.

Fault behavior: If Light Up faults (ignition not confirmed), the AB solenoid closes, AB pump stops and AB igniter turns off immediately. The main engine continues running normally. AB state resets — you can retry from the dashboard.
Light Up Sequence

Runs when afterburner trigger is activated. Default: ABCheckReady → ABSolOpen → ABPumpOn → ABIgnite → ABFlameConfirm → ABStabilize. If empty, the built-in default sequence is used.

Light Off Sequence

Runs when afterburner trigger is released / deactivated. Default: ABSolClose → ABPumpOff.
