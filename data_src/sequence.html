Sequence Editor

Edit startup and shutdown sequences. Each block shows its control structure, condition, and configurable parameters. Sequence order and block parameters are saved together in the engine's ecu_config.json file and applied after reboot.

Control Rules

Threshold rules can monitor configured inputs and command auxiliary outputs during Startup or Running. Rules can also request an orderly shutdown or a logged fault shutdown. Rules are suppressed during Shutdown so they cannot re-enable a fuel output after a cut.

Inputs and outputs not enabled in Hardware are unavailable and cannot be selected. Thresholds use canonical units: deg C, bar, RPM and V.

Fault behavior: If Light Up faults (ignition not confirmed), the AB solenoid closes, AB pump stops and AB igniter turns off immediately. The main engine continues running normally. AB state resets — you can retry from the dashboard.
Light Up Sequence

Runs when afterburner trigger is activated. Default: ABCheckReady → ABSolOpen → ABPumpOn → ABIgnite → ABFlameConfirm → ABStabilize. If empty, the built-in default sequence is used.

Light Off Sequence

Runs when afterburner trigger is released / deactivated. Default: ABSolClose → ABPumpOff.
