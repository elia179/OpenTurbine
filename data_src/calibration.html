Flame Threshold Calibration
1. Make sure no flame is present, then click Capture noise floor (~1 s sampling). Keep the threshold near the igniter-on noise floor so weak light-off flame is still detected.
Threshold = noise floor × multiplier (default 3×). Adjust multiplier if needed, then save.
⚠ Recommended: keep "fire igniter while sampling" ON. Spark noise from the igniter raises the ADC floor. Capturing with the igniter firing sets the threshold high enough so that spark bursts during startup are not mistaken for a real flame — preventing false ignition confirmation. Only disable this if your igniter is not yet installed. “Last run average” is a low-rate reference from the previous engine run; it is not used automatically as the ignition threshold.
