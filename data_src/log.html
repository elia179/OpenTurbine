Logs

Review event history, fault snapshots and detailed per-run session data.

How logging works: Event Log records events and compact periodic snapshots for diagnosis. Session Data records selected values in a detailed CSV only for an engine run. Standby snapshots are disabled by default so an idle ECU does not fill the event log with uneventful readings. Viewing, downloading, and clearing the Event Log require STANDBY (or FAULT) with Standby Snapshot Logging turned off — the ECU refuses flash-heavy reads that would contend with active logging.
