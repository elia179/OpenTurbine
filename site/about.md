lede: Open-source control software for experimental small-turbine projects.
---

OpenTurbine is an MIT-licensed project for supported ESP32 boards. It is intended to make configuration and operating information easier to access for builders who understand turbine hardware and safe test practice, without requiring Git or a firmware toolchain for the normal Windows path.

Visit [GitHub](https://github.com/elia179/OpenTurbine) to review source, releases, and the contribution path.
